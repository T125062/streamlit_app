①生成AIの使用について
本課題の作成にあたり、生成AIを使用した（以下使用内容）
　・アプリ作成におけるエラーの修正
　　　・uniqueの後ろに()を付け忘れていたことによるエラー
　　　・スペルミス、コードの順番による変数エラーの修正
　　　・matplotlibを用いたグラフの日本語対応
　　　・デプロイ時に発生したエラー修正
　・表左端に表示されていた整数列を消去する操作
　・授業で扱っていないUI部品の選定
　・READMEの文章の校閲・修正
　・参考アプリをどのように検索するか
　　　・Dashboardで検索し、参考アプリを選んだ
生成AIは主に、エラー解決とAPI調査の補助として利用した

②データについて
統計表名：生命表
政府統計コード：00450012
データ取得日：2026/01/26
各列の意味
　1列目：年
　2列目：1列目の年における日本人男性の平均寿命（単位：歳）
　3列目：1列目の年における日本人女性の平均寿命（単位：歳）
　4列目：1列目の年における年号
注意点
各年の平均寿命のデータは、0歳（生後0週）における平均余命として収集した

③使用したUI部品
1 st.tabs
　・使用した目的
　　表とグラフの情報を整理し、視認性を向上させるため
　　参考にしたアプリのUIの機能に似せるため
　・使用した所
　　表とグラフの切り替え表示に使用した

2 st.metric
　・使用した目的
　　選択したデータ範囲の男女別平均寿命を強調表示するため
　　データ範囲における平均値の変化を一目で分かるようにした
　・使用した所
　　画面上部に男女別平均寿命（選択データの平均）を表示した

3 st.download_button
　・使用した目的
　　選択範囲のデータをCSV形式で保存できるようにするため
　・使用した所
　　表に表示しているデータのダウンロード機能

④参考にしたアプリ
1 Price Action Dashboard
　・参考にした点
　　表とグラフをタブで切り替える画面構成
　　表データのダウンロード機能
　・自分のアプリへの反映
　　st.tabsにより表とグラフを切り替えられるようにした
　　st.download_buttonでダウンロード機能を実装した。

2 Market Internet Dashboard
　・参考にした点
　　アプリ上部に重要な数値指標を配置する画面構成
　・自分のアプリへの反映
　　st.metricにより平均寿命の数値を画面上部に配置した

⑤注意点
このアプリはWindows環境（MS Gothic）を想定しているため、Mac環境では、グラフに表示される日本語が文字化けする可能性がある

⑥分析結果
表示された見出しおよび折れ線グラフから、1996年から2024年における日本の平均寿命は、男性よりも女性の方が一貫して長いことが分かる。
また、長期的に見ると、年を追うごとに男女ともに平均寿命は緩やかに上昇している傾向が確認できる。
